# Frontend

SvelteKit frontend for AI Live Chat Agent.

## Quick Start

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open browser to the URL shown (usually http://localhost:5173)

## Configuration

Make sure the backend is running on `http://localhost:3001` or update the `API_URL` constant in `src/routes/+page.svelte`.
